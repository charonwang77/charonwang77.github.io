【MarkDown】MarkDown语法

Jul 15, 2021 · 1 min read

一、各级标题

一级标题：# 标题  
二级标题：## 标题  
三级标题：### 标题  
四级标题：#### 标题  
五级标题：##### 标题  
六级标题：###### 标题

二、字体

加粗：**文字**

加粗的文字

删除线：~~文字~~

被删除的文字

斜体：*文字*

斜体文字

三、代码块

代码块：```语言

int a = 0;
int b = 1;
print("%d",a+b);

四、引用

引用：>作者
     >>作者

引用A

引用B

五、分割线

分割线：---
分割线：***

分割的段落A

分割的段落B

六、图片插入

在线图片+本地图片：![图片名字](图片路径) // 图片的名字会在图片加载失败时显示
                                   // 本地图片放在和md同一级目录时，路径 = ../imagename.jpg
